## Objective

How-to guides are short, task-focused recipes for recurring researcher journeys.
Each guide starts with the goal you want to accomplish and gives you the steps to do it.
They assume you have completed the [quickstart](#/02-quickstart) and have a working superRA setup.

If you want to understand *why* a mechanism works, see the [Concepts](#/03-concepts) section.
If you need exact field syntax or CLI flags, see the [Reference](#/05-reference) section.

**Guides in this section:**

- [Install and Set Up](#/04-how-to/01-install-and-set-up) — get superRA running on Claude Code, Codex, or another harness
- [Plan a Project](#/04-how-to/02-plan-a-project) — scope and decompose research work into a task tree
- [Work With Task Files](#/04-how-to/03-work-with-task-files) — read, edit, and hand off work via `task.md` files
- [See Your Work (Dashboard)](#/04-how-to/04-see-your-work) — monitor progress with the live dashboard
- [Integrate and Ship](#/04-how-to/05-integrate-and-ship) — protect results, sync with main, and open a PR
- [Resume and Revise](#/04-how-to/06-resume-and-revise) — pick up interrupted work and revise a task tree mid-flight
