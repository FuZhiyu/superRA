## Objective

You get task files and reports that read alike across the whole tree instead of each agent inventing its own conventions: source files cited as line-anchored links you can follow to the exact line, LaTeX math that renders in the dashboard, and consistent tables.

It is the shared style guide every agent follows when it writes a task file or a status report, and you can load it directly for any markdown you want held to the same standard — citation form, math, table layout, figures, and the dashboard-safe subset of raw HTML. See [`report-in-markdown`](skills/report-in-markdown/SKILL.md) for the conventions in full.
