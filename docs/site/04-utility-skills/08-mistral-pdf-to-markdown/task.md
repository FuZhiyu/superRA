## Objective

You get a PDF converted to clean Markdown with image extraction via the Mistral OCR API, which handles the scanned or complex-layout documents that plain text extraction mangles.

It is the conversion step behind the [Zotero reader](#/04-utility-skills/07-zotero-paper-reader) and works standalone too. See [`mistral-pdf-to-markdown`](skills/mistral-pdf-to-markdown/SKILL.md).
