## Objective

Form value-weighted portfolios on the momentum deciles from `02-construct-factors`, compute each decile portfolio's monthly return series, and estimate the winner-minus-loser spread (decile 10 minus decile 1). Report the average monthly spread. The planted momentum premium is positive, so recent winners should out-earn recent losers.

## Results

The winner-minus-loser spread (decile 10 − decile 1) averages **0.58% per month** over the 228-month sample, recovering the planted momentum premium with the expected sign.

| Portfolio | Avg. monthly return (%) |
|---|---:|
| Decile 1 (losers) | 0.52 |
| Decile 10 (winners) | 1.10 |
| Spread (10 − 1) | 0.58 |

The spread is regenerated by `analysis/momentum_sort.py`, reading `data/panel_factors.parquet`.

## Review Notes

> 1. [MAJOR] The portfolios are equal-weighted, but the project convention is value-weighted. ([analysis/momentum_sort.py:22](analysis/momentum_sort.py#L22)) Equal-weighting overweights the small firms inside each momentum decile, so this spread is partly the size premium in disguise. Re-estimate with value weights and report whether the momentum spread survives.
> 2. [MINOR] The results table omits the sample period; add the month range so the reader can see the burn-in was dropped.

REVISE — the headline spread is computed on the wrong weighting scheme; the value-weighted re-estimate is needed before this can be trusted.
